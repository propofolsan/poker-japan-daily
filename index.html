PokerGO Cup 2026 完全決着 — Lichtenberger がメインイベント制覇、総合王者は Brock Wilson

3月中旬にラスベガスで開催された2026 PokerGO Cupが全日程を終了した。メインイベント（$15,000 NLH、参加者52名）では、Andrew "LuckyChewy" Lichtenberger がファイナルテーブルで全員をバストアウトする圧巻のパフォーマンスで優勝。賞金 $257,400 と自身9度目のPGTタイトルを獲得した。決勝ハンドではSoverelのK-Jに対しA-10でコール、フロップでテンヒット、リバーでエースを引き完勝。

シリーズ総合ではBrock Wilson がリーダーボードを制し、PokerGO Cupトロフィーと $25,000 PGTパスポートを手にした。

イベント 優勝者 賞金
Event #7 ($10K NLH) Tony Gregg $173,075
Event #9 Sean Winter $210,000
Event #10 Main Event A. Lichtenberger $257,400
シリーズ総合 Brock Wilson トロフィー + $25K PGT Passport

注目すべきはTony Greggの復活。2013年のWSOPワンドロップ・ハイローラー優勝以来、一線を退いていた同氏がEvent #7で初のPGTタイトルを獲得。Darren Eliasとの ICMチョップ後にトロフィーを勝ち取った。またSean Winterは Event #9 で通算8度目のPGTタイトルを達成している。

WSOP Europe 2026、来週プラハで開幕 — €10M保証メインイベントに注目

WSOP Europe 2026 が 3月31日〜4月12日 の日程でチェコ・プラハの Hilton Prague にて開催される。King's Casino Prague との提携による今年の WSOPE は、全 15ブレスレットイベント を実施。

目玉は何と言っても €5,300 バイインのメインイベント。前回からバイインを引き下げたにもかかわらず、保証賞金プールは €10,000,000（約16.5億円） と大幅に増額された。より参加しやすい価格帯でありながら過去最大規模の保証額となり、ヨーロッパ最大のポーカーフェスティバルとしての地位を確固たるものにしている。

今年から導入されたWSOPグローバル・プレイヤーオブザイヤーレースでは、プラハ（WSOPE）、夏のラスベガス（WSOP）、バハマ（WSOP Paradise）の3大会を統合した「トリプルクラウン」スタイルのシーズンが展開される。年間ポイント王には賞金 $1,000,000 が贈られる。日本人プレイヤーの参戦にも期待したい。

Venetian DeepStack Showdown、$2.2M保証で3月末まで開催中

ラスベガス・ザ ヴェネチアンで開催中の DeepStack Showdown（2月25日〜3月31日）が佳境を迎えている。全51イベント、総保証賞金プール 約$2,200,000 のシリーズは、$800 NLH（$500K保証）、$1,600 NLH（$500K保証）など、ミドルステークスプレイヤーにも参加しやすいラインナップが揃う。

3月末の残りイベントは中〜ハイバイインが中心。ラスベガス滞在中のプレイヤーは、来週開幕のWSOPEに向けたウォームアップとしても活用できるだろう。

2026年の日本人プレイヤー — WSOPE プラハに期待

2025年に歴史的な活躍を見せた日本人プレイヤーたち。小原 順（みさわ）はトリトン・済州シリーズで5度の入賞・優勝を果たし合計 $2,229,000 以上 を獲得、WSOP では4位で $879,939 を手にした。

大谷選手も2023年のWSOPパラダイス優勝（約4.2億円）を筆頭に、2025年トリトン済州では4位（$447,000）を記録。

2026年は来週の WSOP Europe プラハ が最初の大舞台。€5,300のメインイベントは日本人プレイヤーにとっても参加しやすい設定であり、ブレスレット獲得の期待がかかる。また、JOPTのオンラインサテライトを通じた新世代プレイヤーの台頭にも注目だ。
